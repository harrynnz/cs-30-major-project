# Project Description
Similar to the game Wordle, where you have to guess what 5-letters word is correct. You have a total of 6 tries, if it runs out you lost. 
## Needs to Have
-5x6 grid to put in words

-When press a letter, it goes into the grid

-When Backspace, deletes a letter in the grid

-When press Enter, checks if the word or letters is correct

-Display color of grid when checking word (Gray grid for wrong letters, yellow grid for correct letters but not in the right position, green grid for correct letters in correct position)

-Have atleast 5 words in word bank

-"Acceptable" UI
## Nice to Have
-Grid rotate animation

-Shake animation on the column

-Virtual keyboard on screen, and can actually interact with

-Also display color (Green, yellow, gray) on this keyboard so that players could have a better visualization instead of having to check every single letter in each grid

-Clean and smooth UI

-A big word bank or word bank take words from other sources or dictionary

-Check if the word you put in is actually a word (not some random letters together)

-Sound effect when:
    +The word put in is invalid (Not a word)
    +The word put in is correct
